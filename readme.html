Description of Application ================= 'Tindar' is a Dartmouth specific dating app. It parodies the popular application 'Tinder', in it's purpose (to connect users romantically). However, it also parodies 'LinkedIn', by replacing standard terms with business-related ones. For example, each user is called an 'applicant'. And, instead of 'liking' a profile, one gives an 'offer'. 'Tindar' differentiates itself with a few unique features.
